Component to search for flights

In this section we will implement the FlightSearchComponent. Follow the step by step descriptions or do it on your own.

  1. Create a folder entities in your project's app folder.

  2. In this folder, create a file called flight.ts:

    export interface Flight {
        id: number;
        from: string;
        to: string;
        date: string;
        delayed?: boolean;
    }
  1. Create a folder flight-search in src/app for the FlightSearchComponent with the following files:
  • flight-search.component.ts
  • flight-search.component.html
  1. Open the file flight-search.component.ts and assign the selector flight-search:

Implement Component logic and connect it with the template

  1. Create following methods and properties for the FlightSearchComponent:
// flight-search/flight-search.component.ts

    @Component({
        selector: 'flight-search',
        templateUrl: './flight-search.component.html'
    })
    export class FlightSearchComponent implements OnInit {

        from: string;
        to: string;
        flights: Flight[] = [];
        selectedFlight: Flight;

        constructor(private http: HttpClient) {
        }

        ngOnInit() { }

        search(): void {
            // Implementation follows
        }

        select(f: Flight): void {
            this.selectedFlight = f;
        }
    }
  1. Implement the method search. It should use the properties from and to. Use the injected service HttpClient to search for flights and assign the result to the property flights.
Show code 

// flight-search/flight-search.component.ts

import {[...], HttpHeaders, HttpParams} from '@angular/common/http';

 [...]

search(): void {
    const url = 'http://www.angular.at/api/flight';
    const params = new HttpParams()
                        .set('from', this.from || '')
                        .set('to', this.to || '');

    this.http
        .get<Flight[]>(url, {params})
        .subscribe(
        (flights: Flight[]) => {
            this.flights = flights;
        },
        (errResp) => {
            console.error('Error loading flights', errResp);
        });
}

Use *ngIf, *ngFor directives and (click) bindings.

Show code 

<!-- flight-search/flight-search.component.html -->

    <div class="card">

        <table class="table table-contensed" *ngIf="flights.length > 0">
            <thead>
            <tr>
                <th>Id</th>
                <th>From</th>
                <th>To</th>
                <th>Date</th>
                <th></th>
            </tr>
            </thead>
            <tr *ngFor="let f of flights"
                [class.active]="f === selectedFlight">
                <td>{{f.id}}</td>
                <td>{{f.from }}</td>
                <td>{{f.to }}</td>
                <td>{{f.date | date:'dd.MM.yyyy HH:mm'}}</td>
                <td>
                    <a class="btn btn-link" (click)="select(f)">Select</a>
                </td>
            </tr>
        </table>

    </div>
